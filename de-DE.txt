OVERLAY_MODE_XBOX_LONG				Der Game Bar-Modus wird von der Windows Game Bar unterstützt. It doesn't require Admin provileges (as Real Overlay), but you will need to install our Game Bar extension from the Windows Store. Inkompatibel mit Vulkan- und OpenGL-Vollbildspielen.

OVERLAY_MODE_HOOKS_LONG				Dieser Modus ist mit jedem Grafikmodus von Spielen kompatibel. Er funktioniert wie das Steam-Overlay und kann mit DirectX 9, 10, 11, 12, Vulkan und OpenGL-Grafiken verwendet werden. Einige Anti-Cheat-Programme mögen jedoch solche Overlays (DLL-Injektionen) nicht.
OVERLAY_MODE_HOOKS_LAUNCH			Starte einfach das Spiel, warte ein paar Sekunden, und du solltest dein Fadenkreuz sehen.
